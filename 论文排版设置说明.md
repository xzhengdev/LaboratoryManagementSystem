## 7 一键应用 Word 格式脚本

已提供脚本：

```powershell
lab\scripts\Apply-ThesisWordFormat.ps1
```

使用前提：

| 项目 | 要求 |
| --- | --- |
| 系统 | Windows |
| 软件 | 本机已安装 Microsoft Word |
| 文件格式 | `.docx` 或 `.doc` |

默认不覆盖原文件，会生成 `原文件名_格式化.docx`：

```powershell
powershell -ExecutionPolicy Bypass -File .\lab\scripts\Apply-ThesisWordFormat.ps1 -Path .\lab\论文.docx
```

指定输出文件：

```powershell
powershell -ExecutionPolicy Bypass -File .\lab\scripts\Apply-ThesisWordFormat.ps1 -Path .\lab\论文.docx -OutputPath .\lab\论文_最终版.docx
```

直接覆盖原文件：

```powershell
powershell -ExecutionPolicy Bypass -File .\lab\scripts\Apply-ThesisWordFormat.ps1 -Path .\lab\论文.docx -Overwrite
```

脚本会自动设置：

| 内容 | 自动处理 |
| --- | --- |
| 页面 | A4、纵向、页边距、页眉页脚距离、左侧装订线 |
| 页眉 | “中央民族大学本科生毕业论文（设计）”，宋体五号居中，字间距加宽 0.5 磅 |
| 页脚 | 居中页码，宋体五号 |
| 标题 | 一级、二级、三级标题字体、字号、加粗、对齐和行距 |
| 正文 | 中文宋体、英文 Times New Roman、小四、固定 22 磅行距 |
| 图表题 | 宋体五号、居中、加粗 |
| 表格 | 表格居中，表格正文使用宋体/Times New Roman 五号 |
| 参考文献 | 小四、固定 22 磅行距、段前段后 0 行 |
