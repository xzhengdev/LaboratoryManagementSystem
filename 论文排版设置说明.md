# 信工本科生毕业论文（设计）排版设置说明

本文档用于将 `论文框架.md` 转为 Word 后进行格式设置。Markdown 文件只能保存正文结构、标题层级、图表编号和参考文献格式，页边距、页眉页脚、字体字号和行距需要在 Word 中设置。

## 1 页面设置

| 项目 | 设置 |
| --- | --- |
| 纸张 | A4，纵向 |
| 页边距 | 上 2.5 cm，下 2.5 cm，左 2 cm，右 2 cm |
| 装订线 | 位置：左；宽度：0.5 cm |
| 页眉 | 1.8 cm |
| 页脚 | 2 cm |
| 页眉内容 | 中央民族大学本科生毕业论文（设计） |
| 页眉格式 | 宋体，五号，居中，字间距加宽 0.5 磅 |
| 页脚内容 | 页码 |
| 页脚格式 | 宋体，五号，居中 |

## 5 参考文献格式

参考文献已按照 GB/T 7714—2015 进行整理，保留了 DOI 或在线访问地址。排版时参考文献部分按正文格式设置即可。
